World Wide Web

En la informatica, la World wide web(WWW) o red informatica mundial comunmente conocida como la web, es un sistema de distribucion de documentos de hipertexto o hipermedios interconectados y accesibles via internet. Con un navegador web, un usuario visualiza sitios web compuestos de paginas Web que pueden contener texto, imagenes, videos u otros contenidos multimedia, y navega a traves de esas paginas usando hiperenlaces.
